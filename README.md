# Project-Atlas
Project Atlas is a strategic initiative designed to transition GlobalBank from reactive climate compliance to proactive, scenario-driven risk management. With 36 central banks now running active climate stress tests that carry real capital implications, the urgency to act has never been greater.

This platform serves as both an executive presentation tool and a strategic dashboard — providing the Executive Committee with real-time visibility into climate risk exposure, regulatory alignment, and capital reallocation progress.
